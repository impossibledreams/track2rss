Fedex Meter Request Form

Part of the track2rss Project (track2rss.sf.net)
This page utilizes Javascript ONLY and does not connect to any other servers besides Fedex. This form relies on the use of the XmlHttpRequest object which is only supported by some browsers (Internet Explorer, Safara 1.2, FireFox and Mozilla Suite). If you are using a Mozilla-based browser, you may see a security warning. You must allow this page permissions to access the Fedex system.

This HTML form allows you to request a Fedex meter number from their systems. This meter number is required in order to use the track2rss scripts for Fedex tracking. Once you have received a meter number, please copy it and enter it into the appropriate field inside the track2rss script. YOU MUST HAVE AN EXISTING FEDEX ACCOUNT. Additionally, Fedex has two systems used for tracking, testing and production. In order to obtain access to the production systems, you must call Fedex at 800-810-9073.
Please fill this form out (all fields in bold are required). For state and province, use two character abbreviations for the USA. Once filled out, press the submit button to get the results. Copies of the input and output XML packets will be provided in the textboxes below the form. Please see this document for more information.
